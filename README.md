# 乐淘淘电商--前端开发部分

## 项目依赖

* node
* webpack v1
* webpack-dev-server v1.16.5

## 项目运行
* 安装nodejs环境
* 全局安装webpack v1.x
* 全局安装webpack-dev-server v1.16.5
* 项目根目录执行`npm install`初始化项目
* 启动该项目（windows运行环境）
	* 开发模式：`npm run dev`
	* 生产模式：`npm run dist`
* 开发模式下访问预览项目
	* 访问：http://localhost:8088/dist/view/index.html
